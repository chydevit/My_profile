# Project Context

## Purpose
Personal portfolio website showcasing web development skills, projects, and professional experience. The portfolio serves as:
- Professional online presence for potential employers and clients
- Platform for sharing technical blog posts and tutorials
- Showcase of completed projects with case studies
- Central hub for professional contact and networking

## Tech Stack
- Next.js 16.1.1 with App Router
- TypeScript 5+ for type safety
- Tailwind CSS 4 for styling
- Framer Motion 12 for animations
- MDX for blog content
- Resend for email functionality
- Vercel for deployment and analytics
- next-themes for dark mode management
- React Hook Form + Zod for form processing

## Project Conventions
- Use TypeScript strict mode with no `any` types
- Follow Next.js App Router conventions
- Use Tailwind CSS utility classes over custom CSS
- Organize components by type (ui/, layout/, sections/, features/)
- Store content in JSON/MDX files for easy editing
- Follow accessibility best practices (WCAG 2.1 AA)
- Maintain Lighthouse performance score >90
- Use semantic HTML and proper heading hierarchy

## External Dependencies
- Node.js 18+ runtime
- npm or pnpm package manager
- Resend API for contact form emails
- Vercel for hosting and deployment
- Google Fonts for typography (Inter, Outfit)
